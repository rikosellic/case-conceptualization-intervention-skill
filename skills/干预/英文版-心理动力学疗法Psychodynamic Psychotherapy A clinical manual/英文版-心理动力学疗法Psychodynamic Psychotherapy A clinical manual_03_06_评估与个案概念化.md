## 第 3 章：Creating a Safe Place and Beginning the Evaluation

### 章节主线

- 心理动力学治疗有四个阶段：**评估（evaluation）→ 导引/开始（induction/beginning）→ 中期（mid-phase）→ 结束（termination）**。
- 评估阶段的两大任务：(1) 创造一个让来访者感到安全并能谈论极私密事物的环境；(2) 了解来访者是谁以及为什么现在来求助。
- 即使来访者被"转介"来做心理动力学治疗，治疗师也必须完成独立评估，而不是默认心理动力学治疗适合所有人。

### 创造安全环境（Creating a Safe Place）

- **提供抱持环境（holding environment）**——创造条件让来访者感到安全、有保障和信任，这是治疗联盟的基础。
- **共情和非评判态度**：从开放式问题开始，前 5-10 分钟跟随来访者的引导以真正理解主诉。所有话题都可以讨论。
- **关注身体舒适**：干净安静的房间、适当距离的椅子、关掉手机、调节温度。
- **确保保密性**：明确告知保密原则，防止会话期间被打断。
- **展现理解**：将初步印象反馈给来访者，使其感到被倾听和确认。
- **设定框架和边界**：告知来访者你是谁、会谈多长时间、这是心理治疗评估。
- **保持专业和全面**：温暖但不熟悉。这是一段单向关系。

### 评估内容

| 评估维度 | 内容要点 |
|---------|---------|
| **DSM诊断** | 首先建立诊断。情绪和焦虑障碍最常见。物质滥用和躯体问题不可忽视。严重症状可能需要先支持后揭蔽 |
| **病史** | 现病史（从最后一次正常功能开始）、既往史、发展/个人史（早期气质、童年症状、早期关系质量、教育/职业/关系史） |
| **自我功能** | 能否与治疗师建立关系、耐受强烈感受和焦虑、准确感知现实、控制冲动、延迟满足（详见第 4 章）|
| **心理悟性** | 来访者是否能将心理视为有无意识元素。可通过尝试性解释（trial interpretation）评估 |
| **自我反思能力** | 能否"跳出"自身思考来批判性地审视自己的行为 |
| **问题优先级** | 对自我或他人的潜在暴力优先于所有其他问题 |
| **动机** | 来访者是否认为心理治疗可能对自己有帮助 |
| **资源与社会矩阵** | 社会支持、经济状况、时间、保险 |

### 评估心理悟性和自我反思的示例问题

- **心理悟性**："你认为家庭中发生的事情是否影响了你现在的人际关系？"
  - 有心理悟性者：能做出连接或至少思考可能性 → 适合揭蔽技术
  - 缺心理悟性者："为什么他们的问题会影响我？" → 适合支持技术
- **自我反思能力**："你能反思一下——有哪些可能的方式是你自己在导致现在的困难？"
- **对治疗的期望**："你想象的会谈是什么样的？""你对我有什么期望或想法？"

## 第 4 章：Assessment of Ego Function

### 章节主线

- 自我功能（ego function）= 人们管理内在心理生活和与外界关系的方式。**自我强度（ego strength）**是指在内外刺激冲击下仍能保持平衡的能力。
- 评估自我功能对于判断来访者能否从揭蔽无意识材料中获益至关重要。揭蔽需要足够的自我强度来支撑治疗过程、限制退行、维持现实检验和日常功能。
- 低适应性的防御（基于分裂的防御）主导时，提示自我功能薄弱，应以支持技术为主。

### 11 项自我功能

| 功能 | 定义 | 受损表现 |
|------|------|---------|
| **现实检验与现实感（Reality testing/Sense of reality）** | 区分真实与非真实感知；将外部事件和自身体验为真实和熟悉 | 幻觉、妄想、现实解体、人格解体、梦样状态、与他人融合感 |
| **判断力（Judgment）** | 意识到行为的适当性和可能的后果，并据此行事 | 知道危险但不采取相应行动（如知道不安全仍不采取保护措施）|
| **与他人关系/客体关系（Relationships/Object relations）** | 建立稳定、亲密、有爱、分享、共情的关系，将他人视为完整和独立的个体 | 孤立、无法耐受分离、缺乏共情、操纵他人 |
| **感觉刺激调节（Sensory stimulus regulation）** | 主动抵抗过多感官刺激（外部噪音/内部疼痛） | 被人群或噪音淹没，需要完全安静的环境 |
| **情感/焦虑耐受性（Affect/Anxiety tolerance）** | 耐受和调节焦虑及其他强烈正面/负面情绪 | 容易被自身情绪打乱、快速剧烈波动 |
| **冲动控制（Impulse control）** | 以受控的方式行动或疏导感受/冲动 | 暴食、物质滥用、冲动性行为、自伤、乱发脾气 |
| **玩耍能力/服务于自我的退行（Capacity for play）** | 放松、进入幻想和白日梦而不感到焦虑或过度刺激 | 被幻想压倒、难以从退行中恢复 |
| **自我意识/心理悟性（Self-awareness）** | 对自己的内在状态感到好奇、准确识别情感、理解无意识动机 | 对探索行为缺乏兴趣、大量"我不知道"的回答 |
| **自尊调节/准确自我评估（Self-esteem regulation）** | 对自我打击的调节能力；主观能力感与实际能力的匹配程度 | 自尊完全依赖外界认可 |
| **认知功能（Cognitive functions）** | 智力、记忆、注意力、线性思维、抽象思维、问题解决 | 连接经验、识别模式、调和矛盾态度的能力受限 |
| **防御机制（Defenses）** | 无意识且自动的应对压力和冲突的方式 | 见下文分类 |

### 防御机制分类

#### 低适应性防御（Less adaptive）——基于分裂

基于**分裂（splitting）**，常在**客体恒常性（object constancy）**——即认识到好坏可以共存于同一人——未达成时主导。以极大代价保护来访者免受负面感受的冲击。

| 防御 | 特点 | 示例 |
|------|------|------|
| **分裂（Splitting）** | 将好坏感受分到不同人身上 | 来访者理想化母亲，诋毁父亲 |
| **投射（Projection）** | 将不可接受的想法感受视为源自外部 | 被劈腿后不愤怒，反认为女友在散布谣言 |
| **投射性认同（Projective Identification）** | 将感受投射到他人并诱导其体验该感受 | 被忽视晋升后迟到一周，诱导老板愤怒解雇他 |
| **病理性理想化与贬低** | 分裂的自然结果 | 本周认为治疗师完全理解她，下周认为治疗师是骗子 |
| **否认（Denial）** | 否认不可接受感受的存在 | 脸上有巨大肿瘤却说只是"痘痘" |
| **解离（Dissociation）** | 切断自我与当前现实某些方面的连接 | 被打时进入无痛状态，后来丈夫吼叫时也会如此 |
| **行动化（Acting out）** | 通过行动表达无意识感受 | 法语不及格后酒后与陌生人发生关系 |
| **退行（Regression）** | 退回到更早期的功能方式 | 面对考试压力退回父母家让父母照顾 |

#### 较高适应性防御——基于压抑

基于**压抑（repression）**，将不可接受的想法或感受全部或部分保持无意识。适度的压抑防御在日常功能中很常见。

| 防御 | 特点 | 示例 |
|------|------|------|
| **情感隔离（Isolation of affect）** | 压抑情感，保留想法 | 刚被妻子离开却毫无感受 |
| **理智化（Intellectualization）** | 用过度思考取代痛苦感受 | 开始治疗前读了十本心理治疗书 |
| **合理化（Rationalization）** | 为问题情境找合理理由 | 被解雇后告诉妻子这是最好的安排 |
| **置换（Displacement）** | 将感受的对象换成更舒适的人 | 怕父亲，但转换为怕校长 |
| **躯体化（Somatization）** | 压抑想法/情感，体验为身体感受 | 妻亡纪念日前胃痉挛 |
| **抵消（Undoing）** | 反向操作"挽回"不可接受的感受 | 整天坑人，然后给乞丐一美元 |
| **反向形成（Reaction formation）** | 将不可接受的情感转为完全相反的感受 | 对彻夜啼哭的婴儿过度保护 |
| **认同（Identification）** | 内化他人的方面来处理嫉妒和竞争 | 被姐姐和其朋友排斥后开始听相同音乐 |
| **过度情绪化（Excessive emotionality）** | 压抑想法，保留情绪 | 离婚判决当天无反应，却因杂货店延迟而歇斯底里 |
| **外化（Externalization）** | 将内在冲突体验为外部冲突 | 以为纠结于选哪个男人，实则在纠结是否结婚 |
| **性化（Sexualization）** | 将非性的问题变成性相关 | 情感虚空时与男老师调情 |
| **压抑（Repression）** | 将想法/感受/幻想隐藏出意识 | 离婚后完全忘记报税截止日 |
| **转向自身（Turning against the self）** | 以自我替代他人作为负面感受的靶子 | 生父亲的气却变为自我批评 |

#### 最适应性防御（Most adaptive）

| 防御 | 特点 |
|------|------|
| **幽默（Humor）** | 使用幽默缓解不适想法/感受 |
| **利他主义（Altruism）** | 通过为他人做事来处理痛苦感受 |
| **升华（Sublimation）** | 不适感受直接从无意识到意识以有用形式释放 |
| **压抑（Suppression，注意区分 repression）** | 有意识地决定将想法/感受搁置 |

### 超我功能（Super-ego Function）

- 包括良心和个人理想。评估内疚感的调节：过分严苛 → 不恰当的内疚；发展不足 → 缺乏是非感和对他人的关怀。
- 健康超我功能：有是非感，有足够内疚以善待自己和他人。
- 评估问题："做了/想了那件事后你感觉如何？""你认为自己做错了什么吗？""这件事让你对自己感觉如何？"

### 自我功能评估对治疗的意义

- **自我强度高者**：可以耐受揭蔽技术的痛苦，可以将能量用于思考、爱和游戏而非抵御刺激。适合讨论移情、处理幻想和梦。
- **自我强度低者**：大部分能量用于处理"洪水般"的内外刺激。需要直接帮助他们建立薄弱的自我功能——支持技术为主。
- 自我功能**不是静态的**——压力、疾病、角色转变、自尊打击都会暂时削弱自我功能，要求治疗师灵活切换技术。

## 第 5 章：Formulation: Problem → Person → Goals → Resources Model

### 章节主线

- 个案概念化的核心框架：**问题（Problem）→ 人（Person）→ 目标（Goals）→ 资源（Resources）**。
- 两个来访者可能呈现相似的问题，但如果自我功能水平不同，他们需要不同类型的治疗。评估"人"（基线功能）与评估"问题"同等重要。

### 框架构成

| 维度 | 内容 |
|------|------|
| **问题（Problem）** | 现在促使来访者就诊的困难。来访者可能不清楚自己的问题，最紧急的问题可能不是主诉。需排优先级 |
| **人（Person）** | 基线功能水平：自我功能、防御模式、超我功能、优势和弱点、典型回应他人的方式 |
| **目标（Goals）** | 来访者的希望 + 治疗师评估的需要和可实现性。尽可能协作设定。短期（症状缓解）与长期（性格改变）|
| **资源（Resources）** | 系统因素（治疗师供给、治疗类型、时间）和来访者因素（经济、保险、家庭支持、时间）|

### 样本个案概念化

**Mr H**，45 岁，主诉"和妻子相处不好"。

- **问题**：婚姻冲突。近六个月争吵增多，岳母因重病搬来同住。无情绪或焦虑障碍。
- **人**：
  - 自我功能：优秀——判断力好、长期亲密关系多、能耐受焦虑和强烈情感。7 岁时母亲去世，隐约觉得与此有关（有心理悟性）。
  - 防御：以压抑为基础，多适应性防御。通过运动和木工升华焦虑。情感隔离明显——习惯将强烈感受排除在意识外。
  - 优势：长期工作稳定、亲密关系多、治疗动机强、爱妻子、从工作中获得满足。
  - 弱点：周末偶尔过量饮酒、与感受有些脱节。
  - 超我：发育良好，内疚调节适当。
- **目标**：改善夫妻关系 + 理解母亲去世如何影响他的人际关系。
- **资源**：保险覆盖 50%，自付余款，可利用午休时间前来。

## 第 6 章：Indications for Psychodynamic Psychotherapy

### 章节主线

- 使用问题→人→目标→资源模型决定是否适合心理动力学治疗。
- 不适合的情况：慢性自我功能薄弱——冲动控制差、心理悟性低、情感耐受性差、缺乏信任、慢性精神病性/情绪/焦虑障碍、慢性躯体疾病。

### 指征判断

| 维度 | 适合揭蔽为主 | 适合支持为主 |
|------|-------------|-------------|
| **问题** | 无意识因素导致的人际困难、扭曲的自我感知、适应不良的应对 | 急性危机、慢性自我功能薄弱 |
| **人** | 自我强度好，高心理悟性，好自我反思能力 | 自我功能薄弱，低心理悟性，冲动控制差 |
| **目标** | 长期性格改变、改善自尊/人际关系/防御 | 症状缓解、危机管理、功能改善 |
| **资源** | 时间充裕、经济稳定 | 时间有限或需要密集支持 |

### 无意识因素的线索

- **"我卡住了"**——在职业/关系中感觉卡住，提示无意识冲突
- **"我的生活都很好，除了……"**——只有一个领域有问题，提示可能有特定发展停滞
- **重复的适应不良模式**

### 临床示例

**Mr E**，作家，无法开始第二部小说——害怕被视为"昙花一现"。两个对等的力量在冲突：写作的愿望 vs. 曝光的恐惧。

### Agent 使用建议

- **适用场景**：当 agent 需要评估来访者是否适合心理动力学治疗、选择揭蔽还是支持技术为主、或使用问题→人→目标→资源框架进行个案概念化时。
- **可直接引用**：11 项自我功能评估标准、防御机制三级分类（低/较高/最适应性）、问题→人→目标→资源模型框架、超我功能评估问题。
- **需要结合**：`_01_02.md`中的治疗行动理论——理解"为什么支持"和"为什么揭蔽"的理论基础；后续技术章节根据评估结果选择具体干预。
- **注意**：自我功能评估不是一次性工作——应在整个治疗中持续进行。压力、疾病和自尊打击会暂时削弱自我功能。
