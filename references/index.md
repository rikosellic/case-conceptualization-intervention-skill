# References 使用索引

本目录采用两层资料结构：Markdown 文件是默认核心读书笔记，承担常规任务中替代原书的功能；PDF/DOC/DOCX 原件是二级查询资料，只在用户追问原文细节、模板字段、技术步骤、练习材料或诊断边界时打开。

## 使用顺序

1. 先读本文件，确认资料类别和最合适的核心读书笔记。
2. 读取对应 `*.md` 核心读书笔记，直接完成概念化、模板选择或干预建议。
3. 只有在用户提出细节追问，或摘要无法支持具体操作时，打开对应原始资料。

## 个案概念化理论

- `个案概念化理论/skills/cbt_case_formulation.md`：CBT 取向核心读书笔记，处理症状、情境、自动思维、核心信念、回避和维持循环。
- `个案概念化理论/skills/psychodynamic_formulation.md`：心理动力学核心读书笔记，处理关系模式、防御、情感调节、发展经验、移情和核心冲突。

默认规则：用户没有指定流派时，先根据材料特征在 CBT、心理动力学、家庭/系统、整合式之间选择；常规任务不读取整本理论书。

## 个案概念化模板

- `个案概念化模板/skills/psychodynamic_report_outline.md`：动力学个案报告结构。
- `个案概念化模板/skills/family_supervision_outline.md`：家庭/系统督导汇报结构。
- `个案概念化模板/skills/cbt_session_record_template.md`：CBT 会谈记录和过程整理结构。
- `个案概念化模板/skills/zhongde_case_report_format.md`：中德班取向案例报告结构。

默认规则：模板负责报告呈现，不负责替代理论推理。用户指定模板时优先匹配；未指定时选择最能承载材料重点的结构。

## 诊断

- `诊断/skills/icd11_clinical_descriptions.md`：ICD-11 精神、行为与神经发育障碍临床描述与诊断指南核心读书笔记。

默认规则：诊断资料只用于诊断假设、鉴别方向、风险提示和转介建议；除非用户要求或材料证据充分，不输出确定诊断。
